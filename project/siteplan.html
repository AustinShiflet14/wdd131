Garage Dreams

Site Name

Garage Dreams: This name represents the journey and ambition of car enthusiasts starting their first project build.

Site Purpose

The site provides beginner-friendly advice for building your first project car. It includes parts recommendations, budget planning, tool essentials, common mistakes to avoid, and a community form for asking questions or sharing your progress.

Scenarios

  • First-time car builders looking for guidance
  • Experienced builders wanting to share their knowledge

Color Schema

  • Primary Color: Asphalt Gray (#333333) which will be used for the heading and footer and also main color for all text on the page
  • Secondary Color: #A0C4FF to compliment the primary color, will be used for font color of the headers
  • Accent Color: #F6EEE6 for the main background of the page

Typography

  • Heading Font: Bona Nova SC
  • Body Font: Bona Nova SC
